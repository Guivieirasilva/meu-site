Primeiro Projeto em HTML, CSS e Javascript - Planejamento financeiro.

Projeto | Post no Linkedin | GitHub

O Projeto ajuda o usuario a obter um planejamento financeiro, com base em sua renda mensal, auxilia-o a investir seu dinheiro de forma segura e a controlar suas dispensas, informando-o a quantia que se deve aplicar em cada topico, como necessidades, lazer, objetivos de curto prazo, independencia financeira e caixa. cada tópico mencionado tem um popup, informando qual a ideia do tópico e o percentual de sua renda que foi atrelado a tal tópico. O projeto conta com conceitos de acessibilidade como modo claro e modo escuro.

Minha inspiração para o projeto surgiu na finclass (Plataforma de eduacação financeiro do GRUPO PRIMO) do Breno Perruno - Primeiros Passos, onde o mesmo aborda os primeiros passo que você deve dar para começar a investir
